RoboFab + FontTools + Stuff for FontLab 5.0x on OSX

Special one-click installer for RoboFab, FontTools and Numeric. This will install all dependencies compiled for PPC, and Python2.3.5 the way FontLab 5.0x likes it.
RoboFab Installer for OSX, FontLab 5.0x
FontTools has the BSD license. Check the FontTools project page for more info.

RoboFab + FontTools Combo Deluxe

If you don't have fontTools or don't know what fontTools is, take this version. It will install RoboFab and the appropriate FontTools at the same time. FontTools still has a dependency on an older version of Numeric.
